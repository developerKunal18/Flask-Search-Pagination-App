# Flask Search + Pagination App

A Flask application with search and pagination features.

## Features
- Search notes
- Paginated results
- Query filtering
- SQLAlchemy ORM
- SQLite database

## Technologies Used
- Python
- Flask
- Flask-SQLAlchemy
- HTML

## Installation

pip install flask flask-sqlalchemy

## Run

python app.py

Open browser:
http://127.0.0.1:5000

## Purpose
focuses on search systems and pagination in Flask applications.
